# AI Stock Monitoring

一个用于股票行情监控与信号分析的 Python 项目骨架。

## 项目结构

- `main.py`：项目入口
- `src/ai_stock_monitoring/config.py`：配置加载
- `src/ai_stock_monitoring/monitor.py`：监控核心逻辑
- `tests/`：基础测试

## 快速开始

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
python main.py
```
